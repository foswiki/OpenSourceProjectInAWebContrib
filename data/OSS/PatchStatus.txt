---++ Patch Status
The status of the patch.

| *Name* | *Type* | *Tooltip message* |
| PlannedPatch | option | Patch is planned but not yet coded |
| PatchInProgress | option | Patch is being coded but no files released yet |
| PatchTesting | option | Patch released for testing |
| ReleasedNotIncluded | option | Patch is completed but not included in the main %PROJECTNAME% code |
| ReleasedSnapshot | option | Patch is completed and scheduled for being included in the main %PROJECTNAME% code. It is available in a %PROJECTNAME% snapshot release. |
| ReleasedIncluded | option | Patch is completed and is now included in a released %PROJECTNAME% version |
| PatchCancelled | option | Patch is cancelled (no time to do it, does not work, alternative patch is better, ...) |
