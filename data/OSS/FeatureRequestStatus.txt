---++ Feature Request Status

| *Name* | *Type* | *Tooltip message* |
| New | option | Submitted but noone has decided on it. |
| Discussion | option | Feature is being discussed. Participate by editing the topic. You can simply use the comment field at the bottom. |
| Accepted | option | Feature is accepted. Maybe in a modified form after the discussion. We do not yet know who codes it or who when it will be included. |
| RoadMap | option | There is a plan for when it will be implemented and someone has volunteered to do it. |
| Implemented | option | The feature is implemented and available at least in a snap release. |
| Rejected | option | The request is rejected. This state is also used if the feature is already in %PROJECTNAME% or it is a duplicate of existing request. Project maintainer will always give a reason for rejection. |
